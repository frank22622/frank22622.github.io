如何在Github.io搭建Hexo博客

Posted on 2017-04-14 |

之所以搭建这个博客，起因是之前使用过wordpress搭建个站，觉得有点臃肿繁琐，希望能寻找一个干净清爽的轻量级博客系统。后来发现Hexo很合适，与github.io组合，在上面建立个人profile也是挺不错的。 本篇文章简单介绍一下步骤, 算是起个小头。

1.建立Github repository用于托管

这个非常直截了当，需要注意的是，repository的命名规则必须为：你的用户名.github.io。比方你的用户名是zhangsan, 那repository的名字就是zhangsan.github.io。其余设置默认即可。
